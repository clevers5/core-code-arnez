## *1. Ejercicio Char from ASCII Value*

- **Descripción:**
  Escriba una función get_char() / getChar() que tome un número y devuelva el carácter ASCII correspondiente a ese valor.
  
- Solución

```javascript

function getChar(c){
  return String.fromCharCode(c);
}
```
